## The Problem
Most people waste tokens on vague prompts like:
- "explain transformers"
- "write me a chatbot"
- "help me learn AI"

Vague prompt = vague answer = wasted tokens = poor results.

## The Solution
Prompt Optimizer transforms any weak prompt into a perfectly
structured prompt using the 5-part formula:

| Part | What It Does |
|---|---|
| CONTEXT | Background information the LLM needs |
| ROLE | Who the LLM should act as |
| TASK | Clear step-by-step breakdown |
| FORMAT | Exactly how to structure the output |
| CONSTRAINTS | Length, style, what to avoid |

## Features
- Paste any weak prompt
- Get a perfectly structured prompt back
- See quality score before and after (out of 10)
- View exactly what improvements were made
- Side by side comparison of responses
